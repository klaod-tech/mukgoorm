# 먹구름 봇 커맨드 목록

## 접두사 커맨드 (bot.py — 메인봇)

| 커맨드 | 권한 | 설명 |
|--------|------|------|
| `!setup` | 관리자 | #다마고치 채널에 시작 버튼 고정 메시지 전송 |
| `!소환` | 누구나 | 내 다마고치 embed를 쓰레드에 새로 소환 |
| `!sync` | 봇 오너 | 슬래시 커맨드 수동 동기화 (개발용) |

## 슬래시 커맨드

### 온보딩 (bot.py — cogs.onboarding)

| 커맨드 | 권한 | 설명 |
|--------|------|------|
| `/start` | 메시지 관리 | 채널에 온보딩 시작 메시지 전송 |
| `/리셋 [유저]` | 메시지 관리 | 유저의 쓰레드 전체 삭제 + DB 초기화 후 재온보딩 가능 상태로 만들기 |

### 체중관리

슬래시 커맨드 없음 — 버튼 및 설정에서 처리
- ⚖️ 체중 기록 버튼 → Modal 입력
- ⚙️ 내 정보 → 현재 체중 / 목표 체중 변경

### 메일 (bot_mail.py — cogs.email_monitor)

| 커맨드 | 권한 | 설명 |
|--------|------|------|
| `/이메일설정` | 누구나 | 네이버 메일 연동 설정 |
| `/발신자추가` | 누구나 | 알림 받을 발신자 이메일 등록 |
| `/발신자목록` | 누구나 | 등록된 발신자 목록 확인 |
| `/발신자삭제 <이메일>` | 누구나 | 등록된 발신자 삭제 |
