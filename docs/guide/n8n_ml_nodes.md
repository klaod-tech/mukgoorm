## ⚙️ 로직 요약

### 로짓(θ)이란?
- 카테고리별 선호도 점수 (숫자). 초기값 0.
- 높을수록 그 카테고리를 좋아한다는 의미.
- `user_preference_logits` 테이블에 유저별로 7개 행 저장.

### LOGIT_CATEGORIES
```
['한식', '중식', '양식', '분식', '일식', '디저트', '기타']
```

### 업데이트 규칙

| 이벤트 | 해당 카테고리 로짓 변화 |
|--------|----------------------|
| 🏆 월드컵 승리 | +0.5 |
| 💀 월드컵 탈락 | -0.3 |
| 👍 붐업 | +0.2 |
| 👎 붐다운 | -0.2 |
| 🍽 메뉴 실제 선택 | +0.1 |
