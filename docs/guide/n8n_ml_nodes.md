## ⚙️ 로직 요약 (n8n에 무엇을 하는가)

작업 전에 전체 그림을 이해하고 시작하세요.

### 로짓(θ)이란?
- 카테고리별 선호도 점수 (숫자). 초기값 0.
- 높을수록 그 카테고리를 좋아한다는 의미.
- `user_preference_logits` 테이블에 유저별로 7개 행 저장.

### 업데이트 규칙

| 이벤트 | 해당 카테고리 로짓 변화 |
|--------|----------------------|
| 🏆 월드컵 승리 | +0.5 |
| 💀 월드컵 탈락 | -0.3 |
| 👍 붐업 | +0.2 |
| 👎 붐다운 | -0.2 |
| 🍽 메뉴 실제 선택 | +0.1 |

### Softmax 정렬 원리
```
로짓 → 시간 감쇠(γ=0.95/일) → Temperature Softmax(T=1.5) → Dirichlet Floor(ε=0.1) → 확률 → 식당 정렬
```

## ❗ 주의사항

- **Step 0을 반드시 먼저** 실행하세요. 테이블 없이 노드를 추가하면 실행 시 에러납니다.

- 워크플로우가 **Inactive** 상태면 웹훅이 동작하지 않습니다. Active 토글 확인.
- 코드 노드의 코드는 **전체 선택(Ctrl+A) → 붙여넣기** 방식으로 교체하세요.
