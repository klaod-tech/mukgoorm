## Step 3-3 — n8n: /webhook/food/select 추가

`먹구름_봇v2` 캔버스 빈 공간에 아래 흐름 추가:

```
메뉴 선택 입력(Webhook - /food/select)
    → 방문 기록 저장(Supabase - Create, restaurant_log)
    → 식당 카테고리 조회(Supabase - Get, restaurants)
    → 메뉴선택 현재 로짓(Supabase - Get Many, user_preference_logits)
    → 메뉴선택 로짓 계산(Code - logit +0.1)
    → 로짓 업데이트(Supabase - Update, user_preference_logits)
    → 선택 완료 출력(Respond to Webhook)
```

---

### React에서 보내는 데이터 형태

```json
{
  "user_id": "유저UUID",
  "restaurant_id": "식당UUID",
  "menu_name": "돈까스",
  "keywords": ["돈까스"],
  "location": "아산",
  "date": "2026-05-20"
}
```

---

### 메뉴 선택 입력 — Webhook

| 항목 | 값 |
|------|-----|
| **Method** | POST |
| **Path** | `food/select` |
| **Response Mode** | Using 'Respond to Webhook' Node |

### 방문 기록 저장 — Supabase Create

| Field | Value |
|-------|-------|
| `user_id` | `={{ $('메뉴 선택 입력').item.json.body.user_id }}` |
| `restaurant_id` | `={{ $('메뉴 선택 입력').item.json.body.restaurant_id }}` |
| `date` | `={{ $('메뉴 선택 입력').item.json.body.date }}` |

### 식당 카테고리 조회 — Supabase Get

- Operation: `Get`
- Table: `restaurants`
- Filter: `id = {{ $('메뉴 선택 입력').item.json.body.restaurant_id }}`

> `restaurants.category`는 `text[]` 배열. Code 노드에서 `[0]`으로 첫 번째 값 사용.

### 메뉴선택 현재 로짓 — Supabase Get Many

- Table: `user_preference_logits`
- Return All: ✅
- Filter: `user_id = {{ $('메뉴 선택 입력').item.json.body.user_id }}`

### 메뉴선택 로짓 계산 — Code

```javascript
const body = $('메뉴 선택 입력').first().json.body
const category = $('식당 카테고리 조회').first().json.category?.[0] || '기타'

const ALPHA = 0.1
const allLogits = $('메뉴선택 현재 로짓').all()
const existing = allLogits.find(i => i.json.category === category)

return [{
  json: {
    user_id: body.user_id,
    category,
    logit: Math.round(((existing?.json.logit ?? 0) + ALPHA) * 1000) / 1000,
    sample_count: (existing?.json.sample_count ?? 0) + 1,
    updated_at: new Date().toISOString()
  }
}]
```

### 로짓 업데이트 — Supabase Update

- Table: `user_preference_logits`
- **Condition**: `user_id = {{ $json.user_id }}`, `category = {{ $json.category }}`
- **Fields**: `logit`, `sample_count`, `updated_at`

### 선택 완료 출력 — Respond to Webhook

| 항목 | 값 |
|------|-----|
| **Respond With** | JSON |
| **Response Body** | `={{ JSON.stringify({ ok: true, category: $json.category }) }}` |
