## Step 3-1 — n8n: B, C 경로에 menus 추가

> A경로는 이미 `menus: r.menus` 포함되어 있음 ✅

### B: 응답 포맷 노드 수정

`식당 메뉴 연동1`이 이미 각 식당에 `menus` 배열을 합쳐뒀음.  
B 경로에서 `matched` 필터 시 menus를 포함해서 반환하면 됨.

```javascript
const d = $('의도 분류').first().json
const keyword = d.keyword
const allRestaurants = $('식당 메뉴 연동1').first().json.restaurants

const matched = allRestaurants.filter(r =>
  r.menus && r.menus.some(m =>
    (m.menu_name && m.menu_name.includes(keyword)) ||
    (m.tags && m.tags.some(t => t.includes(keyword)))
  )
)

return [{ json: {
  path: 'B',
  message: matched.length > 0
    ? `"${keyword}" 메뉴를 하는 식당들이에요!`
    : `"${keyword}" 메뉴를 하는 식당을 찾지 못했어요.`,
  restaurants: matched.map(r => ({
    restaurant_id: r.id,
    food_name: r.name,
    category: r.category,
    location: r.address,
    description: r.description,
    phone: r.phone,
    link: r.link,
    menus: r.menus || [],          // ← 추가
    reason: `${keyword} 메뉴를 판매해서`
  })),
  keyword: d.keyword,
  user_id: d.user_id
} }]
```

### C: 응답 포맷 노드 수정

enriched 배열에 `menus` 추가:

const enriched = (parsed.recommendations || []).map(r => {

## Step 3-3 — n8n: /webhook/food/select 추가

`먹구름_봇v2` 캔버스 빈 공간에 아래 흐름 추가:

```
메뉴 선택 입력(Webhook - /food/select)
    → 방문 기록 저장(Supabase - Create, restaurant_log)
    → 메뉴선택 현재 로짓(Supabase - Get Many, user_preference_logits)
    → 메뉴선택 로짓 계산(Code - +0.1)
    → 로짓 업데이트(Supabase - Update, user_preference_logits)
    → 선택 완료 출력(Respond to Webhook)
```

### 메뉴 선택 입력 — Webhook

| 항목 | 값 |
|------|-----|
| **Method** | POST |
| **Path** | `food/select` |
| **Response Mode** | Using 'Respond to Webhook' Node |

### 방문 기록 저장 — Supabase Create

| Field | Value |
|-------|-------|
| `user_id` | `={{ $('메뉴 선택 입력').item.json.body.user_id }}` |
| `restaurant_id` | `={{ $('메뉴 선택 입력').item.json.body.restaurant_id }}` |
| `date` | `={{ $now.setZone('Asia/Seoul').toFormat('yyyy-MM-dd') }}` |

### 메뉴선택 현재 로짓 — Supabase Get Many

- Table: `user_preference_logits`
- Return All: ✅
- Filter: `user_id = {{ $('메뉴 선택 입력').item.json.body.user_id }}`

### 메뉴선택 로짓 계산 — Code

```javascript
const raw    = $('메뉴 선택 입력').first().json
const body   = raw.body || raw
const userId   = body.user_id
const category = body.category

if (!category) {
  return [{ json: { skipped: true } }]
}

const ALPHA = 0.1
const allLogits = $input.all()
const existing  = allLogits.find(i => i.json.category === category)
const currentLogit = existing ? (existing.json.logit ?? 0) : 0
const currentCount = existing ? (existing.json.sample_count ?? 0) : 0

return [{
  json: {
    user_id:      userId,
    category:     category,
    logit:        Math.round((currentLogit + ALPHA) * 1000) / 1000,
    sample_count: currentCount + 1,
    updated_at:   new Date().toISOString()
  }
}]
```

### 로짓 업데이트 — Supabase Update

- Table: `user_preference_logits`
- **Condition**: `user_id = {{ $json.user_id }}`, `category = {{ $json.category }}`
- **Fields**: `logit`, `sample_count`, `updated_at`
